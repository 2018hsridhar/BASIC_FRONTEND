# globalCounter.html

This folder contains `globalCounter.html`, a tiny single-page counter demo.

How to open the page

- Open in your default browser (double-click the file or run):
```bash
open /Users/harisridhar/PERSONAL_CODEBASES/BASIC_FRONTEND/HTML_WEB_PAGES/globalCounter.html
```

- Or serve the folder locally and open in a browser (recommended for some browsers):
```bash
cd /Users/harisridhar/PERSONAL_CODEBASES/BASIC_FRONTEND/HTML_WEB_PAGES
python3 -m http.server 8000
# then open http://localhost:8000/globalCounter.html
```

What the page does

- Displays a numeric counter and three buttons: Increment, Decrement, Reset.
- Buttons update the counter immediately in the browser (no server required).

Notes

- The file was fixed to correct malformed HTML and to attach JavaScript event listeners.
- No backend or network connection is required to run the demo.
